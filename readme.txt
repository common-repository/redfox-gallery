== Description ==

RedFox Gallery is a powerful image gallery plugin with a list of advanced options for creating responsive image galleries with beautiful layouts. You can create many types of galleries and customize them according to you. RedFox gallery plugin contains multiple types of layouts and different hover effects.

Using [RF_GALLERY id=XX] shortcode, you can publish Redfox Gallery in any Page or Post in your WordPress sites.

== Requirements == 
* PHP 5.3
* Wordpress installed


### Plugin Features

Plugin settings are very easy and user-friendly. create awesome galleries with different types of layout.


* Responsive Design
* 10 Types Gallery Layouts
* Multiple Image Uploader
* Multiple Shortcode
* 3 Column layouts
* Different Hover effects
* Unlimited colour scheme
* Hide/Show Gallery Title 
* Left/Center/Right Gallery Title Align
* Gallery Title Unlimited Color 
* Hide/Show Image Title 
* Left/Center/Right Image Title Align
* Image Title Unlimited Color
* Image Title URL
* Image Link Open In New Tab Option
* Unique Settings for Each Gallery
* Shortcode Button On Post or Page
* Drag and Drop Image Position
* Unlimited Gallery Background Color 
* Unlimited Border color
* 1 Lightbox 
* Shadow Effects
* Custom CSS Option

== Installation ==

1. Go to `Plugins` > `Add New` > search for `redfox-gallery`
2. Press `Install Now` for the redfox-gallery plugin
3. Press `Activate Plugin`

Manual installation:

1. Upload the `redfox-gallery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
